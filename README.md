# Hisoka Push 🃏

Distribuição mágica de conteúdo em massa para WordPress com agendamento via REST API e integração Yoast SEO.

## Instalação

```bash
npm install
```

## Rodar localmente

```bash
npm run dev
```

Acesse em `http://localhost:5173`

## Build de produção

```bash
npm run build
```

## Stack

- **React 18** + **TypeScript**
- **Vite 5**
- **WordPress REST API v2**
- **Yoast SEO** (via meta fields)
